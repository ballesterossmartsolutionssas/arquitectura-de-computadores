# Arquitectura de Software

Repositorio local para organizar material de la asignatura y entregas del grupo.

## Contenido

- `entrega memoria 32 bits grupo 3/`
  Archivos `.sim1` de la práctica de simulación de memoria de 32 bits, separados por etapas.
- `trabajo circuito 3/`
  Material previo del circuito 3, incluyendo archivos de SimulIDE y el documento de apoyo.
- `Grabación 2026-03-13 195920.mp4`
  Grabación asociada al trabajo.

## Práctica: Memoria de 32 bits

La carpeta `entrega memoria 32 bits grupo 3/` contiene:

- `Grupo_3_Memoria_32_bits_Etapa_1.sim1`
  Visualización de los dos dígitos hexadecimales de escritura a partir del teclado y el selector de display.
- `Grupo_3_Memoria_32_bits_Etapa_2.sim1`
  Almacenamiento en 2 registros de 8 bits con selección de lectura y escritura de 1 bit.
- `Grupo_3_Memoria_32_bits_Etapa_3.sim1`
  Expansión a 4 registros con selección de lectura y escritura de 2 bits y uso de decodificador.

## Uso

1. Abrir los archivos `.sim1` con SimulIDE.
2. Verificar cada etapa por separado.
3. Usar este repositorio para mantener versiones del trabajo y la documentación asociada.

## Notas

- Este repositorio fue inicializado localmente para organizar la entrega.
- Los archivos binarios como `.docx` y `.mp4` se conservan dentro del repo como respaldo del trabajo.
